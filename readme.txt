Easily clone posts, pages, and custom post types with one click.

== Description ==
Simple Clone is a lightweight plugin that allows you to quickly and easily clone posts, pages, and custom post types within your WordPress website. This plugin is perfect for content creators who need to duplicate existing content without re-creating it manually.

The cloned content is set as a draft by default, so you can edit and make changes before publishing. Simple Clone integrates seamlessly into your WordPress dashboard, adding a “Clone” option to the post and page actions.

### Features:
* Clone posts, pages, and custom post types.
* Cloned content is saved as a draft for further editing.
* Works with custom post types created by other plugins.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/simple-clone` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to your posts, pages, or custom post types, and you'll see a "Clone" link under each item.

== Frequently Asked Questions ==

= How do I clone a post or page? =
After activation, go to the posts or pages list in WordPress. You will see a "Clone" link under each item. Click it, and the content will be cloned and saved as a draft.

= Does this plugin work with custom post types? =
Yes, Simple Clone works with any custom post types that follow WordPress standards.

= Can I edit the cloned post? =
Yes, once the post or page is cloned, it will be saved as a draft. You can edit it just like any other post before publishing.
